# AutoEmply Client

React + Vite client for:

- image/PDF upload
- LayoutSpec JSON generation
- Delphi ZIP export
- prompt preset management
- report template browsing

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment

- `VITE_API_BASE_URL`
- `VITE_API_TIMEOUT_MS`
